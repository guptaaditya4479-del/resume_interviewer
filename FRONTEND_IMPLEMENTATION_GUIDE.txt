===================================================================================
    🐛 TROUBLESHOOTING
===================================================================================

ISSUE: "npm: command not found"
SOLUTION: Install Node.js from https://nodejs.org (v18+ recommended)

ISSUE: "Failed to start interview"
SOLUTION:
• Check backend is running: python main.py (port 8000)
• Check backend has CORS enabled
• Open browser console (F12) to see exact error

ISSUE: "Port 3000 already in use"
SOLUTION:
• Kill process on port 3000: npx kill-port 3000
• Or change port in vite.config.js:
  server: { port: 3001 }

ISSUE: Blank page or white screen
SOLUTION:
• Open browser console (F12) for errors
• Check npm run dev is running
• Verify no compilation errors in terminal

ISSUE: "Cannot find module 'react'"
SOLUTION: Run npm install in frontend directory
